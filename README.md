# teleirc.hs

Simple Telegram <-> IRC relay bot written in Haskell

Requirements:
------
* haskell
* Modules:
    * http-conduit

Running (with interpreter)
------
```
ghci main.hs
```
And then just type `main`

Running (compiled executable)
------
```
ghc --make main.hs
./main
```
(Or ./main.exe in Windows)
